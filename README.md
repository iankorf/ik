The ik Library
==============

Toolbox for building simple sequence analysis programs. How simple is simple?
Nucleic acid and amino acid sequences are simply `char *`.

## Core Functions ##

+ Toolbox
	+ Dynamic arrays
	+ Maps (hashes)
	+ Error output
	+ Command line
+ Sequence
	+ FASTA files
	+ Utility functions
